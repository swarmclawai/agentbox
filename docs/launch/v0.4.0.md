# agentbox v0.4.0

## Headline

Agentbox now has a local run library.

## Release Notes

`@swarmclawai/agentbox@0.4.0` makes recorded runs easier to revisit after the first replay. You can list runs, open the latest replay, generate a searchable offline library page, compare two runs, and safely clean old artifacts.

New commands:

```bash
agentbox list
agentbox library --open
agentbox open latest
agentbox compare <base> <head>
agentbox clean --keep 10 --dry-run
```

Run targets now support `latest`, exact ids, unique id prefixes, run directories, and paths to files inside a run.
