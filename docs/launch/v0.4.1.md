# agentbox v0.4.1

## Headline

Agentbox now has launch-ready demo scenarios.

## Release Notes

`@swarmclawai/agentbox@0.4.1` adds deterministic demo scenarios for success, failure, and MCP risk review.

Each scenario creates the full local evidence bundle:

```bash
agentbox demo --scenario success
agentbox demo --scenario failure
agentbox demo --scenario mcp-risk
```

The output includes:

- self-contained `agentbox-run.html`
- Markdown `REPORT.md`
- redacted export zip
- file diffs, terminal replay, MCP events, risk flags, and redaction metadata
